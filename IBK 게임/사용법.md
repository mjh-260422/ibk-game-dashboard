# IBK 게임 대시보드 자동 생성 가이드

엑셀 로우데이터를 넣으면 PDF 대시보드를 자동으로 만들어주는 스크립트입니다.

---

## 1. 사전 준비 (처음 한 번만)

### Python 설치 확인
터미널(명령 프롬프트)에서 아래 명령어 실행:
```
py --version
```
버전이 나오면 OK. 없으면 https://python.org 에서 설치.

### 필요 패키지 설치
```
py -m pip install pandas matplotlib openpyxl
```

---

## 2. 폴더 구성

```
IBK게임/
├── generate_dashboard.py   ← 스크립트
├── 사용법.md               ← 이 파일
└── 게임결과목록xxxx.xlsx   ← 여기에 로우데이터 넣기
```

---

## 3. 로우데이터 엑셀 파일 조건

파일명은 아무거나 상관없음. 아래 컬럼이 있어야 함:

| 컬럼명 | 설명 |
|--------|------|
| 사용자 ID | 참여자 식별값 |
| 게임명 | 게임 이름 |
| 게임 실행일 | 날짜+시간 형식 |
| 차감 포인트 | 숫자 |
| 경품명 | 당첨 경품 이름 |
| 경품 지급 상태 | `완료` 또는 그 외 |

---

## 4. 실행 방법

### 방법 A — 파일 자동 탐색 (가장 간단)
`generate_dashboard.py`와 같은 폴더에 xlsx 파일 하나만 두고 실행:
```
py generate_dashboard.py
```

### 방법 B — 파일 직접 지정
```
py generate_dashboard.py 게임결과목록2604240948.xlsx
```
또는 전체 경로:
```
py generate_dashboard.py C:\Users\이름\Downloads\게임결과목록.xlsx
```

---

## 5. 출력 결과

같은 폴더에 `IBK_게임_대시보드.pdf` 생성됨.

**대시보드 구성 (16:9, 1페이지)**
- KPI 카드 5개: 총 실행 수 / 유니크 사용자 / 총 차감 포인트 / 지급 완료 / 지급 미완료
- 게임별 실행 수 막대 차트
- 경품별 당첨 수 가로 막대 차트

---

## 6. 클로드 코드에서 실행하는 경우

Claude Code 터미널에서:
```
cd C:\Users\이름\IBK게임
py generate_dashboard.py
```

또는 클로드에게 그냥 말로 요청:
> "게임결과목록 파일로 대시보드 만들어줘"
