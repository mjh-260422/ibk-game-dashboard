# IBK 게임 운영현황 보고 사용 가이드

## 전체 흐름

```
① 어드민에서 파일 다운로드
        ↓
② 보고생성.bat 실행 (파일 드래그)
        ↓
③ 구글 시트에서 [보고 생성] 클릭
        ↓
④ 내부보고 / 외부보고 확인
```

---

## 1단계: 파일 다운로드

사내 어드민에서 3개 파일을 다운로드합니다.

| 파일 | 형식 | 설명 |
|------|------|------|
| 로우데이터 | `.xlsx` / `.xls` | 게임 참여 전체 데이터 (누적, 서비스 시작일부터 전부) |
| 할인쿠폰 | `.csv` | 할인쿠폰 발행 목록 |
| 매체사 경품 | `.csv` | 매체사 경품 발행 목록 |

> ⚠️ **로우데이터는 서비스 시작일(2026-04-23)부터 전체 기간 데이터**를 받아야 합니다.

---

## 2단계: 보고생성.bat 실행

`보고생성.bat`을 더블클릭하면 모드 선택 화면이 뜹니다.

```
실행 모드를 선택하세요:
  1. 전체 처리 (첫 실행 또는 전체 재처리)
  2. 일별 추가 (전일 데이터 누적)
```

| 상황 | 선택 |
|------|------|
| 서비스 시작 이후 처음 실행하거나 데이터 전체 재처리 | **1** |
| 매일 전일 데이터 추가 업로드 | **2** |

이후 파일 경로를 입력합니다 (파일을 창에 드래그하면 자동 입력).

```
로우데이터 엑셀 경로: → 파일 드래그 또는 직접 입력
할인쿠폰 CSV 경로: → 파일 드래그 (없으면 엔터)
매체사경품 CSV 경로: → 파일 드래그 (없으면 엔터)
```

> ⚠️ **모드 1 (전체)**: 로우데이터는 서비스 시작일부터 전체 누적 데이터  
> ✅ **모드 2 (일별 추가)**: 전일 하루치 데이터만 있으면 됩니다

---

## 3단계: 구글 시트에서 보고 생성

1. [IBK 게임 운영현황 구글 시트](https://docs.google.com/spreadsheets/d/1G2A_FyERvQOVQBUsu9AHx7FPrE-UAIX4Ohl9UNtalzY) 열기
2. 상단 메뉴 **IBK 게임 → 보고 생성** 클릭
3. 잠시 기다리면 **내부보고 / 외부보고** 시트가 업데이트됨

## 자주 묻는 질문

### Q. 오류가 났어요
- `패키지 설치 오류`: 터미널에서 `py -m pip install pandas google-auth google-api-python-client openpyxl` 실행
- `파일을 찾을 수 없습니다`: 파일 경로에 한글 폴더가 있으면 바탕화면 같은 곳으로 파일 이동 후 재시도
- `인증 오류`: `~/.claude/google-sheets-key.json` 파일이 있는지 확인, 없으면 담당자(명지혜)에게 문의

### Q. 보고서 항목을 추가하거나 수정하고 싶어요
Claude Code에서 아래처럼 요청하면 됩니다:
> "내부보고에 ○○ 항목 추가해줘"
> "일별 현황에 ○○ 컬럼 넣어줘"

GAS 코드(`운영현황보고.gs`)만 수정하면 됩니다. Python 스크립트는 건드릴 필요 없어요.

### Q. 집계 결과를 직접 확인하고 싶어요
구글 시트에서 `[집계]` 로 시작하는 시트들을 열어보면 Python이 계산한 중간 결과를 볼 수 있습니다.

| 시트명 | 내용 |
|--------|------|
| `[집계]누적` | 총 실행수, 참여자, 포인트 등 전체 합계 |
| `[집계]게임별` | 게임별 실행 횟수 |
| `[집계]게임경품` | 게임별 경품 발행 건수 |
| `[집계]경품사용` | 경품별 발행/사용/만료 현황 |
| `[집계]쿠폰` | 할인쿠폰 종류별 발행/사용 |
| `[집계]월별` | 월별 집계 |
| `[집계]일별` | 일별 집계 |

```bash
py -m pip install pandas google-auth google-api-python-client openpyxl
```
